# Saigon Guacamole Website

A simple static website for Saigon Guacamole menu and ordering.

## Project Structure

```
saigon-guac-site/
├── index.html          # Landing page
├── menu.html           # Menu page
├── assets/
│   └── images/
│       ├── SaigonGuac-removebg-preview.png  # Logo
│       └── wmremove-guac.png                # Hero image
└── README.md
```

## Deployment

This site is ready for deployment on Vercel or any static hosting service.

### Vercel Deployment

1. Push this folder to a GitHub repository
2. Connect the repository to Vercel
3. Vercel will automatically detect it as a static site
4. Deploy!

## Local Development

Simply open `index.html` or `menu.html` in your browser to test locally.

## Features

- Pure HTML/CSS (no frameworks)
- Responsive design
- Print-friendly menu
- Optimized images
- SEO-friendly meta tags
